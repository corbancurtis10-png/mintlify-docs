---
title: "Get Started with EZAnim in 5 Minutes"
sidebarTitle: "Quickstart"
description: "Learn how to add EZAnim to your Roblox project, initialize the service with your animation list, and play your first animation in under 5 minutes."
---

This guide walks you through installing EZAnim, setting up your animation list, and playing your first animation on a character model. By the end you will have a working animation service you can use anywhere in your game.

<Steps>
  <Step title="Add EZAnim to your project">
    Place the `EZAnim.luau` module script inside `game.ReplicatedStorage` (or any shared location accessible from both the server and client).

    In Roblox Studio, right-click `ReplicatedStorage` → **Insert Object** → **ModuleScript**, rename it `EZAnim`, then paste in the module source.

    If you use Rojo, add the file to your `src/ReplicatedStorage/` directory and sync.
  </Step>

  <Step title="Require the module and initialize">
    In any Script or LocalScript, require EZAnim and call `init` with your animation ID list:

    ```lua Script (ServerScriptService)
    local AnimationService = require(game.ReplicatedStorage.EZAnim)

    local anim = AnimationService.init({
        List = {
            idle   = "rbxassetid://111111111",
            run    = "rbxassetid://222222222",
            attack = "rbxassetid://333333333",
            jump   = "rbxassetid://444444444",
        }
    })
    ```

    The keys (e.g. `"idle"`, `"run"`) are the names you will use when calling `play`, `stop`, and other methods. The values are Roblox asset IDs for your uploaded animations.
  </Step>

  <Step title="Play an animation on a character">
    Call `anim:play(model, id, type, looped)` to start an animation:

    ```lua
    local Players = game:GetService("Players")

    Players.PlayerAdded:Connect(function(player)
        player.CharacterAdded:Connect(function(character)
            -- Play the idle animation, looped
            anim:play(character, "idle", "Idle", true)
        end)
    end)
    ```

    | Parameter | Type | Description |
    |-----------|------|-------------|
    | `model` | `Model` | The character model (must have a `Humanoid`) |
    | `id` | `string` | Key from your animation `List` |
    | `type` | `string` | Priority: `"Idle"`, `"Movement"`, `"Action"`, `"Action2"`, `"Action3"`, `"Action4"` |
    | `looped` | `boolean?` | Whether the animation loops (default `false`) |
  </Step>

  <Step title="Stop the animation">
    Call `stop` with the model and animation key to stop a specific track, or omit the key to stop all tracks on the model:

    ```lua
    -- Stop a specific animation
    anim:stop(character, "idle")

    -- Stop all animations on the character
    anim:stop(character)
    ```
  </Step>
</Steps>
