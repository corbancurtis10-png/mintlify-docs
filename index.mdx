EZAnim is a Roblox Luau module that wraps Roblox's Animator API into a clean, simple interface. Drop it into any project to manage character animations — play, stop, pause, resume, tween, lerp, and ragdoll — without boilerplate.

## What EZAnim Does

EZAnim gives you a single service object that handles everything animation-related on a Roblox `Model` with a `Humanoid`:

- **Play / Stop / Pause / Resume** — full playback control
- **Priority system** — Idle, Movement, Action, Action2–4
- **Animation caching** — loads each animation once per model
- **Ragdoll & unragdoll** — physics-based ragdoll with optional auto-recovery
- **Tween & Lerp helpers** — built-in `TweenService` and `Vector3` interpolation
- **Event callbacks** — react to animation markers and stop events

## Minimal Example

```lua EZAnim basic usage
local AnimationService = require(game.ReplicatedStorage.EZAnim)

local anim = AnimationService.init({
    List = {
        run  = "rbxassetid://123456789",
        idle = "rbxassetid://987654321",
    }
})

-- Play the "run" animation on a character model
anim:play(character, "run", "Movement", false)

-- Stop it later
anim:stop(character, "run")
```

<Note>
  EZAnim requires a `Humanoid` inside the target `Model`. If no `Animator` exists under the Humanoid, EZAnim creates one automatically.
</Note>
